# DevOps Internship Tasks

## Task 1 — CI/CD with GitHub Actions
Set up a CI/CD pipeline using GitHub Actions to test, build, and push a Docker image to DockerHub on every push to main.

## Task 2 — CI/CD with Jenkins
Replicated the same CI/CD pipeline using a self-hosted Jenkins server on Windows. Used a Jenkinsfile with bat commands and Jenkins credentials store.

## Task 3 — IaC with Terraform
Provisioned a local Docker container using Terraform's Docker provider. Ran terraform init, plan, apply, state list, and destroy.

## Task 4 — Git Workflow
Demonstrated Git best practices including branching, pull requests, tags, .gitignore, and markdown documentation.
